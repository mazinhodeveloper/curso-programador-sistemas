Flex-direction

A propriedade define que a direção do contêiner que empilha os itens flex.

Seus valores:

  • column: empilha os itens flexíveis verticalmente (de cima para baixo).
  • column-reverse: empilha os itens flexíveis verticalmente invertendo sua ordem (de baixo para cima).
  • row: empilhad os itens flexíveis horizontalmente (da esquerda para direita) - PADRÃO.
  • row-reverse: empilha os itens flexíveis horizontalmente invertendo sua ordem (da direita para esquerda).

Flex-wrap

A propriedade especifica se os itens flexíveis devem ser ou não de acordo com o viewport.

Seus valores são:

  • wrap: especifica que os itens flexíveis serão enrolados, se necessario.
  • nowrap: especifica que os itens flexíveis não serão enrolados - PADRÃO.
  • wrap-reverse: especifica que os itens flexíveis irão enrolar se necessários e na ordem inversa.

Justify-content

A propriedade é usada para alinhas os itens flexíveis de forma horizontal.

Seus valores são:

  • center: Os valores de alinhamento flexíveis no centro do contêiner.
  • flex-start: Os valores de alinhamento flexíveis no início do contêiner - PADRÃO.
  • flex-end: Os valores de alinhamento flexíveis no final do contêiner.
  • space-around: Os valores de alinhamento flexíveis com espaço antes, entre e depois.
  • space-between: Os valores de alinhamento flexíveis com espaço entre os objetos flexíveis.

Align-items

A propriedade é usada para alinhar os itens flexíveis na vertical.

Seus valores são:

  • center: O valor alinha os itens flexíveis no meio do contêiner.
  • flex-start: O valor alinha os itens flexíveis na parte superior do contêiner.
  • flex-end: O valor alinha os itens flexíveis na parte inferior do contêiner.
  • stretch: O valor estende os itens flexíveis para preencher o contêiner. - PADRÃO.
  • baseline: O valor alinha os itens flexíveis, com as linhas bases.

Align-content

A propriedade é usada para alinhar as linhas flexíveis.

Seus valores são:

  • center: O valor mostra as linhas flexíveis no meio do contêiner.
  • stretch: O valor estica as linhas flexíveis para assumir o contêiner - PADRÃO.
  • flex-start: O valor exibe as linhas flexíveis no início do contêiner.
  • flex-end: O valor exibe as linhas flexíveis no final do contêiner.
  • space-around: O valor exibe as linhas flexíveis com espaço antes, entre e depois do contêiner.
  • space-between: O valor exibe as linhas flexíveis com espaço entre eles no contêiner.
